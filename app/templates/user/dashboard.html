Quick Start

Use your API key with any OpenAI-compatible client:

export OPENAI_API_KEY="tsk-your-key-here"
export OPENAI_BASE_URL="http://your-server:8000/v1"

Python SDK:

from openai import OpenAI
client = OpenAI(api_key="tsk-...", base_url="http://your-server:8000/v1")
response = client.chat.completions.create(model="qwen-turbo", messages=[{"role":"user","content":"Hello"}])
